## install

###### before install: you create a config.json simple like

```json
    "token": "Your token here",
    "clientId": "your app id here",
    "guildId": ["your server discord list here"],
    "prefix":"your prefix here"
```

###### next step install node modules

use yarn

```yarn
yarn install
```

or npm if you want

```npm
npm install
```

## run

use yarn

```yarn
yarn start
```

or npm if you want

```npm
npm start
```
